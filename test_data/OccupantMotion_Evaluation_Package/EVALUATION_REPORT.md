# 主动控制座椅 vs 被动座椅 — 乘员头部运动响应全维度评测报告

## 1. 数据概况

| 项目 | 值 |
|:---|:---|
| 测试对象 | **真人志愿者** |
| 实验组 | 主动控制座椅 (IMU1头部眉心-1 + IMU9胸骨剑突-1) |
| 对照组 | 被动座椅 (IMU2头部眉心-2) |
| 采集方式 | **同一趟采集**，同时记录两组IMU数据 |
| 采样率 | 1000 Hz |
| 数据时长 | 47.9 秒 |
| 传感器位置 | 头部眉心、胸骨剑突、座垫R点、座椅底部 |

---

## 2. 核心发现

### 2.1 低频段衰减效果显著

| 频段 | Ax | Ay | Az |
|:---|---:|---:|---:|
| **0.1-0.5 Hz** | **54.0%** | **93.9%** | **82.0%** |
| **0.5-1.0 Hz** | **30.3%** | **85.8%** | **48.6%** |
| 1.0-5.0 Hz | — | 31.2% | 15.9% |

**结论：主动控制座椅在0.1-1Hz超低频段（对应车身俯仰/侧倾/弯道）表现出色，尤其横向Ay方向低频衰减高达93.9%。**

### 2.2 制动事件关键指标

| 指标 | 实验组(主动) | 对照组(被动) | 衰减效率 |
|:---|---:|---:|---:|
| Ay RMS | 2.59 m/s² | 8.35 m/s² | **69.0%** |
| Az RMS | 1.45 m/s² | 4.46 m/s² | **67.6%** |
| Ax RMS | 4.92 m/s² | 4.98 m/s² | 1.2% |

**结论：制动减速时，主动座椅对横向晃动(Ay)和垂向颠簸(Az)的衰减效果显著，纵向(Ax)因制动惯性主导，衰减不明显。**

### 2.3 全量统计对比

| 指标 | 实验组(主动) | 对照组(被动) | 改善 |
|:---|---:|---:|---:|
| Ay VDV | 10.20 | 22.27 | **+54.2%** |
| Az RMS | 1.55 | 2.26 | **+31.5%** |
| Ax VDV | 14.03 | 11.94 | -17.5% |

### 2.4 事件分类统计

- 检测到16个驾驶事件
- 1次显著制动减速 (speed: 38→2 km/h)
- 15次复合工况 (加速/减速+转向)
- 转向/变道事件: 6次有效识别

## 3. 分析维度与方法

### 3.1 时域分析
- **滑动窗口分析** (1秒窗, 0.5秒步长): 全时域衰减效率
- **事件窗口分析**: 16个事件逐一精细化对比
- **全量统计**: RMS, Peak, Crest Factor, VDV

### 3.2 频域分析
- Welch功率谱密度 (0.1~80Hz)
- 频段衰减率: 低频0.1-0.5Hz / 0.5-1Hz / 1-5Hz / 5-20Hz / 20-80Hz
- 相干性分析

### 3.3 相关性分析
- Speed/Wheel vs 头部Ay加速度
- 实验组 vs 对照组互相关
