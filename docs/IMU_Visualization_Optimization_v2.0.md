# IMU 专业可视化优化方案 v2.0

## 📋 概述

基于您提供的高采样率数据可视化优化建议，我们开发了全新的专业级IMU可视化组件 `imu_pro_visualizer.py`，解决了1000Hz高采样率数据在有限像素上的渲染问题。

---

## 🔥 核心优化点

### 1️⃣ **视窗内动态降采样 (分段极值采样)** - 优先级1
- **核心算法**: `ExtremaDownsampler` 类
- **实现原理**:
  - 将当前视窗内的数据按屏幕像素数（默认500点）分成N段
  - 每段取最大值和最小值，保留波形的峰谷细节
  - 渲染点数量固定在500个，与屏幕像素匹配
- **三种降采样模式**:
  - `EXTREMA`: 极值采样（推荐，保留峰谷）
  - `AVERAGE`: 平均采样
  - `MAX_POINTS`: 最大点采样

### 2️⃣ **坐标轴与视图配置优化** - 优先级2
- **固定Y轴范围**（默认关闭自适应）:
  - 加速度: ±16 m/s²
  - 角速度: ±10 rad/s
  - 车速: 0-150 km/h
  - 转向角: -720°~720°
- **多通道独立分窗显示**:
  - 通道1: 加速度 (AX, AY, AZ)
  - 通道2: 角速度 (GX, GY, GZ)
  - 通道3: 车速 (Speed)
  - 通道4: 转向角 (Wheel)
- **可配置视窗时长**:
  - 选项: 0.5s, 1.0s, 2.0s, 5.0s, 10.0s, 30.0s, 60.0s
  - 支持分析细节（短窗口）和趋势（长窗口）

### 3️⃣ **视觉呈现细节优化** - 优先级3
- **曲线样式**:
  - 线宽: 1.5~2.0px
  - 高对比度配色: 红/绿/蓝/青/紫/黄/白/橙
  - 透明度优化
- **峰值标记**:
  - 圆点标记关键峰值
  - 便于快速识别关键节点
- **参数瓦片显示**:
  - 实时数值显示
  - 趋势箭头（▲上升/▼下降）
  - 报警色（黄色警告/红色超限）

### 4️⃣ **底层性能与数据预处理** - 优先级4
- **低通滤波**:
  - 实时数据平滑
  - 减少高频噪声
- **数据缓冲**:
  - 环形缓冲区 (maxlen=5000)
  - 批量渲染优化
- **渲染优化**:
  - 30ms定时刷新 (~33 FPS)
  - 抗锯齿渲染

## 🚀 使用指南

### 快速集成
```python
from modules.ui.visualization.imu_pro_visualizer import IMUProVisualizer

# 初始化
imu_pro = IMUProVisualizer()

# 连接数据流
imu_pro.set_data_bridge(data_bridge)

# 启动
imu_pro.start()

# 接收数据 (与现有接口兼容)
sensor_data = {
    't': timestamp,
    'ax': accel_x,
    'ay': accel_y,
    'az': accel_z,
    'gx': gyro_x,
    'gy': gyro_y,
    'gz': gyro_z,
    'speed': vehicle_speed,
    'wheel': steering_angle
}
imu_pro.receive_imu_data(sensor_data)
```
