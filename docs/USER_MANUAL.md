# 用户使用手册

## 📖 **概述**

欢迎使用重构后的右侧面板系统！本系统完全替代了模拟数据和模拟模型，连接到真实的Core系统，为您提供实时数据监控、分析和可视化功能。

## 🎯 **系统特点**

- **真实数据**: 完全替代模拟数据，显示真实的传感器和分析数据
- **实时同步**: 左侧控制面板与右侧显示面板的实时数据同步
- **智能恢复**: 意外中断后自动恢复任务执行
- **模式切换**: 支持重构模式与传统模式的动态切换

## 🚀 **快速开始**

### **启动系统**
```bash
# 标准启动
python main/main.py

# 或者使用集成版本
python modules/ui/core_ui_main_integration.py
```

## 🎮 **左侧控制面板**

### **1. 数据源控制面板**
控制数据源的连接和配置。

#### **主要功能**
- **数据源类型选择**: 选择要监控的数据源类型
  - IMU传感器数据
  - CNAP心血管数据
  - 系统状态数据
  - 自定义数据源

- **连接控制**: 管理数据源连接
  - 连接/断开按钮
  - 连接状态指示器
  - 连接参数配置

- **数据配置**: 设置数据采集参数
  - 采样频率
  - 数据格式
  - 缓存大小

#### **操作步骤**
1. 选择数据源类型
2. 配置连接参数
3. 点击"连接"按钮
4. 监控连接状态

### **2. 解析控制面板**
管理数据解析和预处理。

#### **主要功能**
- **解析器选择**: 选择适合的解析器

  - CNAP数据解析器
  - 心血管数据解析器

- **解析控制**: 控制解析过程
  - 开始/停止解析
  - 解析参数调整
  - 解析状态监控

- **数据验证**: 验证解析后的数据质量
  - 数据完整性检查
  - 异常数据检测
  - 数据格式验证

#### **操作步骤**
1. 选择数据解析器
2. 设置解析参数
3. 启动解析过程
4. 监控解析状态

### **3. 分析控制面板**
控制数据分析和处理。

#### **主要功能**

- **分析控制**: 管理分析过程
  - 启动/停止分析

- **模型管理**: 管理分析模型

4. 查看分析结果

## ⚙️ **高级功能**

### **模式切换**
系统支持两种运行模式：

#### **重构模式 (默认)**
- 使用新的重构后右侧面板
- 完全替代模拟数据
- 支持所有新功能

#### **传统模式**
- 使用传统右侧面板
- 保持向后兼容性
- 基本功能支持

#### **切换方法**
1. 在右侧面板中右键点击
2. 选择"切换到传统模式"或"切换到重构模式"
3. 系统自动重新加载相应面板

### **数据导出**
支持多种数据导出格式：

#### **导出格式**
- **CSV**: 逗号分隔值文件
- **JSON**: JavaScript对象表示法
- **Excel**: Microsoft Excel文件
- **PDF**: 便携式文档格式

#### **导出步骤**
1. 在相应标签页中选择"导出"功能
2. 选择导出格式
3. 设置导出参数
4. 选择保存位置
5. 确认导出

### **自定义配置**
支持用户自定义配置：

#### **配置项目**
- **UI主题**: 选择界面主题
- **语言设置**: 设置界面语言

## 🔧 **故障排除**

### **常见问题**

#### **问题1: 数据不显示**
**症状**: 右侧面板没有数据显示
**可能原因**:
- 数据源未连接
- 解析器未启动
- 数据同步失败

**解决方案**:
1. 检查左侧控制面板的连接状态
2. 确认解析器已启动
3. 查看同步状态
4. 检查系统日志

#### **问题2: 界面响应缓慢**
**症状**: 界面操作响应慢
**可能原因**:
- 数据更新频率过高
- 内存使用过多
- 系统资源不足

**解决方案**:
1. 降低数据更新频率
2. 清理系统缓存
3. 重启系统
4. 检查系统资源使用

#### **问题3: 数据同步失败**
**症状**: 左侧控制与右侧显示不同步
**可能原因**:
- 同步管理器故障
- 网络连接问题
- 配置错误

**解决方案**:
1. 重启同步管理器
2. 检查网络连接
3. 验证配置参数
4. 查看错误日志
