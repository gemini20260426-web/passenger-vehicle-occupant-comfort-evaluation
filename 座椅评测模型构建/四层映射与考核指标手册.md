## 第二层: 位置 → 数据字段

每个IMU提供7个时间序列字段:

| 字段名 | 含义 | 单位 | 符号 |
|--------|------|------|------|
| `Ax_m_s2` | X轴加速度(纵向) | m/s² | a_x |
| `Ay_m_s2` | Y轴加速度(侧向) | m/s² | a_y |
| `Az_m_s2` | Z轴加速度(垂向) | m/s² | a_z |
| `Gx_dps` | X轴角速度(滚转) | °/s | ω_x |
| `Gy_dps` | Y轴角速度(俯仰) | °/s | ω_y |
| `Gz_dps` | Z轴角速度(偏航) | °/s | ω_z |
| `rel_time` | 相对时间 | s | t |
| `channel` | CAN通道 | - | ch1~ch5 |
| `imu_name` | IMU名称(主索引键) | - | |
| `_source_type` | 数据来源类型 | - | can_long |

## 第三层: 数据字段 → 算子

| 算子 | 函数 | 输入字段 | 输出 | 标准依据 |
|------|------|---------|------|---------|
| OP-CFC | `cfc_filter(a, fs, class)` | Ax/Ay/Az | 滤波后加速度 | SAE J211-1 |
| OP-PSD | `welch_psd(a, fs)` | Ax/Ay/Az | G(f) 功率谱密度 | ISO 2631-1 |
| OP-CSD | `cross_psd(x, y, fs)` | Ax,Ay,Az(成对) | G_xy(f) 交叉谱 | ISO 10326-2 |
| OP-WK/WD | `frequency_weighting(psd, f, axis)` | PSD | 加权PSD | ISO 2631-1 |
| OP-STFT | `stft_spectrogram(a, fs)` | Ay/Ax | S(t,f) 时频谱 | ISO 18431-4 |
| OP-SRS | `srs_maximax(a, fs, Q=10)` | Ax/Az | SRS(f) 冲击响应谱 | MIL-STD-810H |
| OP-RAIN | `rainflow_counting(a)` | Ax/Ay/Az | 幅值-均值循环矩阵 | ASTM E1049 |
| OP-INT | `integrate(acc, fs)` | Ax/Ay/Az | 位移(mm) | - |
| OP-ATTEN | `calc_attenuation(imu_exp, imu_ctrl)` | 指标值 | 衰减效率(%) | V4.06手册 |

## 第四层: 算子 → 考核指标

| 指标 | 算子链 | 输入IMU | 单位 | 阈值 |

| HIC15 | OP-CFC→OP-VECSYN→计算 | IMU1/2(头部) | - | ≤650 |
| ACC-PEAK | OP-CFC→峰值检测 | IMU1/2(头部) | g | 魔椅<传统 |
| JERK | 微分→峰值检测 | IMU1/2(头部) | g/s | 魔椅<传统 |
| SRS-MRS | OP-SRS→最大响应 | IMU1/2(头部) | m/s² | 魔椅峰值<传统 |
| SRS-Q | SRS峰值/输入峰值 | IMU1/2(头部) | - | <1.5优良 |

## 使用示例

```python
from seat_evaluation import SeatEvaluator

# 1. 初始化引擎
evaluator = SeatEvaluator()

# 2. 加载实测数据 (从CSV/JSON导入)
evaluator.load_imu_data('IMU7_座椅底部-1', {
    'rel_time': t_array,       # 时间序列
    'Ax_m_s2': ax_array,       # X轴加速度
    'Ay_m_s2': ay_array,       # Y轴加速度
    'Az_m_s2': az_array,       # Z轴加速度
    'Gx_dps': gx_array,        # X轴角速度
    'Gy_dps': gy_array,        # Y轴角速度
    'Gz_dps': gz_array,        # Z轴角速度
})
# ... 加载全部10路IMU ...

# 3. 计算全部指标
results = evaluator.compute_all_indicators()

# 4. 导出
evaluator.print_summary()
evaluator.print_mapping()
evaluator.export_results_json('results.json')
```
