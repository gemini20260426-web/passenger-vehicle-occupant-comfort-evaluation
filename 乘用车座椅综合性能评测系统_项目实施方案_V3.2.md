## 1. 项目概述

### 1.1 系统定位

本系统 **不是** 传统的"车辆驾驶行为分析系统"，而是 **基于驾驶行为事件驱动的乘员生理、生物力学和运动力学评测系统**。

**核心价值主张**：
- 以"驾驶行为事件"（如AEB紧急制动、蛇形驾驶、变道避障等）作为评测触发点
- 针对乘员（头部、躯干、脊柱）的生理响应、生物力学特性、运动力学表现进行多维度量化
- 采用"稳态频域 + 瞬态多维"双轨评价体系，实现对座椅综合性能的科学评测
- **完全整合现有系统的五层分析架构，不重复造轮子**

### 1.2 项目背景

**核心问题**：传统V4.06实验手册基于ISO 2631-1等频域分析方法，在处理瞬态/冲击激励时面临根本性困难：

| 试验工况 | 激励类型 | PSD失效原因 | 科学量化难点 |
|----------|----------|-------------|--------------|
| AEB紧急制动 | 纵向瞬态冲击 | 信号非平稳，能量集中于&lt;1s | 峰值减速度/冲击持续时间的复合效应 |
| 双移线/蛇形驾驶 | 侧向确定性机动 | 频率成分随车速/转角时变 | 侧倾速率与回位速度的瞬态耦合 |
| 减速带/坑洼冲击 | 垂向瞬态冲击 | 非平稳、宽带激励 | 冲击传递率与共振放大的区分 |
| 复合工况（变道+制动） | 多轴瞬态耦合 | 多事件叠加、时序依赖 | 各事件贡献的分离与量化 |

**现有系统分析**：
现有系统已经具备完整的五层驾驶行为分析架构（layer1-layer5），通过pipeline.py进行编排。我们的方案是在这个现有架构基础上进行扩展，而不是重构。

### 1.3 项目目标

**主要目标**：
1. 在现有五层分析架构基础上，构建基于"驾驶行为事件驱动"的统一评测架构
2. 实现"稳态频域 + 瞬态多维"双轨评价体系（共28项考核指标）
3. 落地9类处理算子系统（5类原有 + 4类新增）
4. 建立四维分析方法论框架（STFT + SRS + 雨流计数+FDS + MBS）
5. **完全打通现有系统的五层分析输出与座椅评测引擎**

**量化指标**：
- 支持28项可配置考核指标（13项保留 + 15项新增）
- 事件检测延迟&lt;100ms，指标计算延迟&lt;500ms
- 新增指标开发时间&lt;1人天/单个指标
- 系统可用性≥99%
- **事件到评测的自动触发成功率≥99.5%**

### 2.3 核心编排器 AnalysisPipeline

**文件位置**：`core/core/analysis/pipeline.py`

**核心代码结构**：

```python
class AnalysisPipeline:
    def __init__(self, vehicle_config: Optional[VehicleConfig] = None):
        self._signal_processor = SignalProcessor(self._vehicle_config)
        self._feature_extractor = FeatureExtractor(self._vehicle_config)
        self._state_machine = DrivingStateMachine()
        self._maneuver_detector = ManeuverDetector()
        self._temporal_validator = TemporalConsistencyValidator()
        self._classifier = HybridBehaviorClassifier()
        self._risk_assessor = RiskAssessor(self._vehicle_config)
    
    def process_frame(self, raw_data: Dict[str, Any]) -> FrameResult:
        # ===== Layer 1: 信号处理 =====
        frame = self._signal_processor.process(raw_data)
        
        # ===== Layer 2: 特征工程 =====
        features = self._feature_extractor.extract(frame)
        
        # ===== Layer 3: 机动分割 =====
        state, state_changed, maneuver_ended = self._state_machine.update(frame, features)
        
        event = None
        if maneuver_ended:
            event = self._maneuver_detector.end_maneuver(frame)
            if event and self._temporal_validator.validate(event):
                # ===== Layer 4: 行为分类 =====
                event = self._classifier.classify(event, features)
                
                # ===== Layer 5: 风险评估 =====
                event = self._risk_assessor.assess(event, features)
                self._recent_events.append(event)
        
        if state_changed and state not in (DrivingState.UNKNOWN, DrivingState.STOPPED):
            self._maneuver_detector.begin_maneuver(state, frame)
        
        self._maneuver_detector.update(frame, features)
        
        result = FrameResult(...)
        return result
```

## 3. 多通道IMU数据源与对照分析设计

### 3.1 CAN数据集多通道结构

**数据源配置**：
- **总通道数**：10个IMU通道（imu1-imu10）
- **分组方式**：实验组 vs 对照组
- **数据类型**：IMU传感器数据（加速度、角速度、姿态角）
- **采样频率**：≥100Hz

**通道分组方案**：

| 分组 | 通道ID | 座椅类型 | 测量位置 |
|------|--------|---------|---------|
| **实验组（魔椅）** | imu1-imu5 | 新型座椅 | 头枕、靠背、座垫、头枕旁、地板 |
| **对照组（传统座椅）** | imu6-imu10 | 传统座椅 | 头枕、靠背、座垫、头枕旁、地板 |

### 3.2 对照分析核心需求

**核心目标**：
1. 实时同步采集实验组和对照组数据
2. 基于同一驾驶事件触发双组评测
3. 生成对照分析报告，量化差异

**关键需求**：
- **数据同步**：实验组与对照组数据精确时间对齐（±10ms）
- **事件对齐**：同一事件同时触发两组评测
- **指标对比**：28项指标逐项对比分析
- **差异量化**：统计显著性差异、改善幅度
- **可视化**：双组对比曲线、差异热图、改善看板

### 4.2 核心数据流程

#### 4.2.1 事件驱动评测流程（基于现有五层架构）

```
1. 多通道IMU数据接入 (imu1-imu10)
   ↓
2. 数据同步与对齐 (新增模块)
   ↓
3. 送入AnalysisPipeline (五层处理)
   ↓
4. Layer 1-5 处理完成，输出FrameResult
   ↓
5. 从FrameResult中提取ManeuverEvent
   ↓
6. BehaviorEventDispatcher检查事件是否在映射中
   ↓
7. 如果是，发射comparative_evaluation_trigger (包含双组配置)
   ↓
8. ComparativeEvaluationEngine接收触发
   ↓
9. 分别提取实验组(imu1-5)和对照组(imu6-10)的数据窗口
   ↓
10. 对两组数据分别运行SeatEvaluationEngine
    ↓
11. 对比分析：逐项对比、计算改善、t检验
    ↓
12. 发射comparative_evaluation_result
    ↓
13. CommunicationBus广播结果
    ↓
14. UI更新显示 + 数据持久化存储
```

# -*- coding: utf-8 -*-
"""
多通道IMU数据同步与对齐模块
用于实验组和对照组数据的精确时间对齐
"""

import numpy as np
import logging
from typing import Dict, List, Optional, Any
from collections import deque
from PySide6.QtCore import QObject, Signal

logger = logging.getLogger(__name__)


class DataSyncAligner(QObject):
    """数据同步与对齐器"""
    
    data_aligned = Signal(dict)  # 对齐后的数据
    sync_status_updated = Signal(str, float)  # 同步状态
    
    def __init__(self, config_manager=None):
        super().__init__()
        self.config_manager = config_manager
        
        # 通道分组配置
        self.channels_config = {
            'experimental': ['imu1', 'imu2', 'imu3', 'imu4', 'imu5'],
            'control': ['imu6', 'imu7', 'imu8', 'imu9', 'imu10'],
        }
        
        # 数据缓冲区 (timestamp -> data)
        self.data_buffers: Dict[str, deque] = {
            'experimental': deque(maxlen=1000),
            'control': deque(maxlen=1000),
        }
        
        # 同步精度要求 (ms)
        self.max_time_offset_ms = 10
        
        # 最新对齐的时间戳
        self.last_aligned_timestamp = 0.0
        
        logger.info("数据同步与对齐器初始化完成")
    
    def add_data(self, channel_id: str, data: Dict[str, Any]) -> None:
        """添加单个通道的数据"""
        group = None
        if channel_id in self.channels_config['experimental']:
            group = 'experimental'
        elif channel_id in self.channels_config['control']:
            group = 'control'
        else:
            logger.warning(f"未知通道: {channel_id}")
            return
        
        # 包装数据
        wrapped_data = {
            'channel_id': channel_id,
            'timestamp': data.get('timestamp', 0.0),
            'data': data,
            'group': group
        }
        
        self.data_buffers[group].append(wrapped_data)
        
        # 尝试对齐
        self._try_align()
    
    def add_multi_channel_data(self, multi_data: Dict[str, Dict[str, Any]]) -> None:
        """添加多通道数据"""
        for channel_id, data in multi_data.items():
            self.add_data(channel_id, data)
    
    def _try_align(self) -> Optional[Dict[str, Any]]:
        """尝试对齐数据"""
        # 检查两个组都有数据
        if (len(self.data_buffers['experimental']) == 0 or
            len(self.data_buffers['control']) == 0):
            return None
        
        # 获取最新的时间戳
        exp_last = self.data_buffers['experimental'][-1]['timestamp']
        ctrl_last = self.data_buffers['control'][-1]['timestamp']
        
        # 找到共同的基准时间戳
        if exp_last > ctrl_last:
            target_time = ctrl_last
        else:
            target_time = exp_last
        
        # 如果目标时间戳与上次对齐的一样，跳过
        if target_time <= self.last_aligned_timestamp:
            return None
        
        # 对齐两组数据到目标时间戳
        exp_aligned = self._align_group('experimental', target_time)
        ctrl_aligned = self._align_group('control', target_time)
        
        if exp_aligned and ctrl_aligned:
            # 构建对齐后的数据
            aligned_result = {
                'timestamp': target_time,
                'experimental': exp_aligned,
                'control': ctrl_aligned,
                'time_offset_ms': abs(exp_last - ctrl_last) * 1000
            }
            
            self.last_aligned_timestamp = target_time
            self.data_aligned.emit(aligned_result)
            
            # 更新状态
            offset_ms = abs(exp_last - ctrl_last) * 1000
            status = f"同步成功，偏移: {offset_ms:.2f}ms"
            self.sync_status_updated.emit(status, offset_ms)
            
            return aligned_result
        
        return None
    
    def _align_group(self, group: str, target_time: float) -> Optional[Dict[str, Any]]:
        """对齐单组数据到目标时间戳"""
        buffer = self.data_buffers[group]
        if len(buffer) == 0:
            return None
        
        # 找到最接近目标时间的数据点
        best_match = None
        best_diff = float('inf')
        
        for data in reversed(buffer):
            diff = abs(data['timestamp'] - target_time) * 1000
            if diff < best_diff:
                best_diff = diff
                best_match = data
            if diff < self.max_time_offset_ms:
                break  # 已经足够接近
        
        if best_match and best_diff < self.max_time_offset_ms:
            return best_match
        else:
            return None
    
    def get_aligned_window(
        self,
        start_time: float,
        end_time: float
    ) -> Dict[str, List[Dict[str, Any]]]:
        """获取指定时间窗口内的对齐数据"""
        result = {
            'experimental': [],
            'control': []
        }
        
        for group in ['experimental', 'control']:
            for data in self.data_buffers[group]:
                if start_time <= data['timestamp'] <= end_time:
                    result[group].append(data)
        
        return result
    
    def clear(self) -> None:
        """清空缓冲区"""
        self.data_buffers['experimental'].clear()
        self.data_buffers['control'].clear()
        self.last_aligned_timestamp = 0.0
        logger.info("数据同步缓冲区已清空")

#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
对照分析引擎
用于实验组和对照组的双组评测与对比分析
"""

import numpy as np
import logging
from typing import Dict, List, Optional, Any, Tuple
from scipy import stats
from dataclasses import asdict
from PySide6.QtCore import QObject, Signal

logger = logging.getLogger(__name__)


class ComparativeEvaluationEngine(QObject):
    """对照分析引擎"""
    
    comparative_evaluation_started = Signal(dict)
    comparative_evaluation_completed = Signal(dict)
    test_group_report_ready = Signal(dict)
    comparison_progress = Signal(int, str)  # 进度
    
    def __init__(self, config_manager=None, data_storage=None):
        super().__init__()
        self.config_manager = config_manager
        self.data_storage = data_storage
        
        # 评测引擎实例
        from core.core.seat_evaluation.engine import SeatEvaluationEngine
        self.engines = {
            'experimental': SeatEvaluationEngine(config_manager, data_storage),
            'control': SeatEvaluationEngine(config_manager, data_storage),
        }
        
        # 事件结果收集
        self.event_results: List[Dict[str, Any]] = []
        
        # 统计方法
        self.stats_methods = {
            'ttest': self._independent_ttest,
            'wilcoxon': self._wilcoxon_signed_rank,
        }
        
        logger.info("对照分析引擎初始化完成")
    
    def evaluate_comparative(self, trigger: Dict[str, Any]) -> Optional[Dict[str, Any]]:
        """执行对照评测"""
        self.comparison_progress.emit(0, "开始对照评测")
        self.comparative_evaluation_started.emit(trigger)
        
        try:
            trigger_id = trigger.get('event_id')
            logger.info(f"开始对照评测: {trigger_id}")
            
            # 1. 提取两组数据
            self.comparison_progress.emit(10, "提取实验数据")
            exp_data = self._extract_group_data(
                trigger,
                trigger.get('experimental_channels', ['imu1', 'imu2', 'imu3', 'imu4', 'imu5'])
            )
            
            ctrl_data = self._extract_group_data(
                trigger,
                trigger.get('control_channels', ['imu6', 'imu7', 'imu8', 'imu9', 'imu10'])
            )
            
            if not exp_data or not ctrl_data:
                logger.warning("数据提取失败")
                return None
            
            # 2. 执行两组评测
            self.comparison_progress.emit(30, "执行实验组评测")
            exp_results = self._evaluate_single_group('experimental', trigger, exp_data)
            
            self.comparison_progress.emit(60, "执行对照组评测")
            ctrl_results = self._evaluate_single_group('control', trigger, ctrl_data)
            
            if not exp_results or not ctrl_results:
                logger.warning("评测执行失败")
                return None
            
            # 3. 对比分析
            self.comparison_progress.emit(80, "执行对比分析")
            comparisons = self._analyze_comparison(exp_results, ctrl_results)
            
            # 4. 计算总体评分
            overall_score = self._calculate_overall_score(exp_results, ctrl_results)
            
            # 5. 组装结果
            comparative_result = {
                'trigger_id': trigger_id,
                'event_type': trigger.get('event_type'),
                'source_behavior': trigger.get('source_behavior'),
                'timestamp': trigger.get('timestamp'),
                'experimental_results': exp_results,
                'control_results': ctrl_results,
                'comparisons': comparisons,
                'overall_score': overall_score,
            }
            
            # 6. 保存并发射结果
            self.event_results.append(comparative_result)
            self.comparative_evaluation_completed.emit(comparative_result)
            self.comparison_progress.emit(100, "对照评测完成")
            
            logger.info(f"对照评测完成: {trigger_id}")
            return comparative_result
            
        except Exception as e:
            logger.error(f"对照评测失败: {e}", exc_info=True)
            return None
    
    def _extract_group_data(
        self,
        trigger: Dict[str, Any],
        channels: List[str]
    ) -> Optional[Dict[str, Any]]:
        """提取单组数据"""
        try:
            data_window = trigger.get('data_window', {'pre': 0.5, 'post': 1.5})
            event_time = trigger.get('timestamp')
            
            # 从data_storage或缓冲区提取数据
            if self.data_storage:
                start_time = event_time - data_window['pre']
                end_time = event_time + data_window['post']
                group_data = self.data_storage.get_data_by_channels(
                    channels=channels,
                    start_time=start_time,
                    end_time=end_time
                )
            else:
                # 回退使用raw_data
                group_data = trigger.get('raw_data', {})
            
            return {
                'channels': channels,
                'data': group_data,
                'data_window': data_window,
                'event_time': event_time
            }
        except Exception as e:
            logger.error(f"提取数据失败: {e}")
            return None
    
    def _evaluate_single_group(
        self,
        group: str,
        trigger: Dict[str, Any],
        group_data: Dict[str, Any]
    ) -> Optional[Dict[str, Any]]:
        """执行单组评测"""
        try:
            engine = self.engines[group]
            
            # 创建单组评测触发
            eval_trigger = {
                'event_id': f"{trigger.get('event_id')}_{group}",
                'event_type': trigger.get('event_type'),
                'metrics': trigger.get('metrics', []),
                'timestamp': trigger.get('timestamp'),
                'raw_data': group_data,
            }
            
            # 执行评测
            result = engine.evaluate_by_event(eval_trigger)
            return result
            
        except Exception as e:
            logger.error(f"{group}组评测失败: {e}")
            return None
    
    def _analyze_comparison(
        self,
        exp_results: Dict[str, Any],
        ctrl_results: Dict[str, Any]
    ) -> Dict[str, Dict[str, Any]]:
        """对比分析"""
        comparisons = {}
        
        exp_metrics = exp_results.get('metrics', {})
        ctrl_metrics = ctrl_results.get('metrics', {})
        
        for metric_id in set(exp_metrics.keys()).union(ctrl_metrics.keys()):
            exp_val = exp_metrics.get(metric_id)
            ctrl_val = ctrl_metrics.get(metric_id)
            
            if exp_val is not None and ctrl_val is not None:
                comparisons[metric_id] = self._compare_metric(
                    metric_id,
                    exp_val,
                    ctrl_val
                )
        
        return comparisons
    
    def _compare_metric(
        self,
        metric_id: str,
        exp_val: float,
        ctrl_val: float
    ) -> Dict[str, Any]:
        """对比单个指标"""
        # 计算差异
        diff = exp_val - ctrl_val
        
        # 计算改善百分比
        if abs(ctrl_val) > 1e-6:
            improvement_pct = ((ctrl_val - exp_val) / ctrl_val) * 100
        else:
            improvement_pct = 0.0
        
        # 统计检验
        stat_result = self._independent_ttest([exp_val], [ctrl_val])
        
        return {
            'experimental': exp_val,
            'control': ctrl_val,
            'diff': diff,
            'improvement_pct': improvement_pct,
            'stat_sig': stat_result.get('significant', False),
            'p_value': stat_result.get('p_value', 1.0),
            'effect_size': stat_result.get('cohens_d', 0.0),
        }
    
    def _independent_ttest(
        self,
        sample1: List[float],
        sample2: List[float]
    ) -> Dict[str, Any]:
        """独立样本t检验"""
        try:
            if len(sample1) < 2 or len(sample2) < 2:
                return {'significant': False, 'p_value': 1.0, 'cohens_d': 0.0}
            
            t_stat, p_value = stats.ttest_ind(sample1, sample2, equal_var=False)
            
            # 计算Cohen's d效应量
            n1, n2 = len(sample1), len(sample2)
            s1, s2 = np.std(sample1, ddof=1), np.std(sample2, ddof=1)
            pooled_std = np.sqrt(((n1 - 1) * s1**2 + (n2 - 1) * s2**2) / (n1 + n2 - 2))
            
            m1, m2 = np.mean(sample1), np.mean(sample2)
            
            if pooled_std > 0:
                cohens_d = (m1 - m2) / pooled_std
            else:
                cohens_d = 0.0
            
            return {
                'significant': p_value < 0.05,
                'p_value': p_value,
                'cohens_d': cohens_d,
                't_stat': t_stat
            }
            
        except Exception as e:
            logger.error(f"t检验失败: {e}")
            return {'significant': False, 'p_value': 1.0, 'cohens_d': 0.0}
    
    def _calculate_overall_score(
        self,
        exp_results: Dict[str, Any],
        ctrl_results: Dict[str, Any]
    ) -> Dict[str, float]:
        """计算总体评分"""
        exp_score = exp_results.get('overall_score', 0.0)
        ctrl_score = ctrl_results.get('overall_score', 0.0)
        
        # 计算改善量
        if abs(ctrl_score) > 1e-6:
            improvement = ((ctrl_score - exp_score) / ctrl_score) * 100
        else:
            improvement = 0.0
        
        return {
            'experimental_score': exp_score,
            'control_score': ctrl_score,
            'improvement_pct': improvement
        }
    
    def generate_test_group_report(self, report_name: str) -> Optional[Dict[str, Any]]:
        """生成试验组报告"""
        if len(self.event_results) == 0:
            logger.warning("没有对照评测结果")
            return None
        
        # 汇总统计
        summary_statistics = self._summarize_results(self.event_results)
        
        report = {
            'report_id': f"{report_name}_{int(time.time())}",
            'test_name': report_name,
            'start_time': self.event_results[0].get('timestamp'),
            'end_time': self.event_results[-1].get('timestamp'),
            'event_count': len(self.event_results),
            'event_results': self.event_results,
            'summary_statistics': summary_statistics,
        }
        
        self.test_group_report_ready.emit(report)
        return report
    
    def _summarize_results(
        self,
        event_results: List[Dict[str, Any]]
    ) -> Dict[str, Dict[str, Any]]:
        """汇总结果统计"""
        summary = {}
        
        # 收集所有指标的值
        metric_values: Dict[str, Dict[str, List[float]]] = {}
        
        for result in event_results:
            for group in ['experimental', 'control']:
                group_results = result.get(f"{group}_results", {})
                metrics = group_results.get('metrics', {})
                
                for metric_id, value in metrics.items():
                    if metric_id not in metric_values:
                        metric_values[metric_id] = {'experimental': [], 'control': []}
                    metric_values[metric_id][group].append(value)
        
        # 计算统计
        for metric_id, values in metric_values.items():
            exp_vals = values['experimental']
            ctrl_vals = values['control']
            
            if len(exp_vals) > 0 and len(ctrl_vals) > 0:
                exp_mean = np.mean(exp_vals)
                exp_std = np.std(exp_vals, ddof=1)
                ctrl_mean = np.mean(ctrl_vals)
                ctrl_std = np.std(ctrl_vals, ddof=1)
                
                if abs(ctrl_mean) > 1e-6:
                    improvement_pct = ((ctrl_mean - exp_mean) / ctrl_mean) * 100
                else:
                    improvement_pct = 0.0
                
                # 总体t检验
                stat_result = self._independent_ttest(exp_vals, ctrl_vals)
                
                summary[metric_id] = {
                    'exp_mean': exp_mean,
                    'exp_std': exp_std,
                    'ctrl_mean': ctrl_mean,
                    'ctrl_std': ctrl_std,
                    'improvement_pct': improvement_pct,
                    'p_value': stat_result.get('p_value', 1.0),
                    'significant': stat_result.get('significant', False),
                    'count': len(exp_vals)
                }
        
        return summary
    
    def clear(self) -> None:
        """清空结果"""
        self.event_results.clear()
        logger.info("对照分析结果已清空")

### 6.2 四维分析方法论框架

#### 6.2.1 STFT（短时傅里叶变换）- 时频分析
- **用途**：分析非平稳信号的时变频率成分
- **适用场景**：蛇形驾驶、变道等瞬态机动
- **输出指标**：
  - STFT-FC（频率中心）
  - STFT-KT（频率扩展）
  - STFT-CE（能量集中度）

#### 6.2.2 SRS（冲击响应谱）- 冲击分析
- **用途**：分析冲击激励对座椅的影响
- **适用场景**：AEB紧急制动、减速带冲击
- **输出指标**：
  - SRS-MRS（最大响应谱）
  - SRS-Q（品质因数）
  - SRS-PV（峰值速度）
  - SRS-ATT（衰减时间）

#### 6.2.3 雨流计数+FDS - 疲劳分析
- **用途**：累积损伤计算
- **适用场景**：长期驾驶、颠簸路面
- **输出指标**：
  - RFC-CC（循环计数）
  - FDS-D（损伤度）
  - FDS-R（剩余寿命）

#### 6.2.4 MBS（多体仿真）- 结构分析
- **用途**：座椅结构受力分析
- **适用场景**：优化座椅设计
- **输出指标**：
  - DISP-TR（位移轨迹）

### 6.3 28项考核指标体系

#### 6.3.1 原有13项指标（保留）
| 指标ID | 指标名称 | 单位 | 评价维度 |
|-------|---------|------|---------|
| SEAT-Z | 座椅垂直加速度 | g | 频域 |
